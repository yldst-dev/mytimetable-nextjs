# PWA 아이콘 생성

이 디렉토리는 PWA 아이콘들을 포함합니다.

## 개발용 아이콘 생성
현재는 개발용 SVG 기반 아이콘을 사용하고 있습니다.
프로덕션 환경에서는 다음을 권장합니다:

1. 전문 디자인 도구로 고품질 아이콘 제작
2. PWA 아이콘 생성기 사용 (예: https://www.pwabuilder.com/)
3. 모든 플랫폼과 크기에 최적화된 아이콘 생성

## 아이콘 크기
- 72x72px
- 96x96px
- 128x128px
- 144x144px
- 152x152px
- 192x192px
- 384x384px
- 512x512px
